1. Положить папку templates в src/main/resources Spring Boot проекта
2. Описание шаблонов:
	а) signup.html - страница регистрации
	б) main.html - главная страница (входная точка приложения)
3. В шаблонах присутствует только html-верстка
4. Необходимо заполнить пропуски в шаблонах, отмеченные "...":
	а) ссылки на методы контроллеров, которые обрабатывают GET/POST-запросы
	б) значения в полях text
	в) значения в полях selected
	и т.д. комментарии, какой элемент за что отвечает, даны в html-шаблонах
5. Скриншоты с работающими шаблонами приложены для примера, как должно выглядеть
